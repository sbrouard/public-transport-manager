La réalisation avec le lien à-un se trouve dans le repertoire a-un.
La réalisation avec le lien est-un se trouve dans le repertoire est-un.

Ces répertoires contiennent:

-Un répertoire build qui contient les fichiers compilés.
-Un fichier log.txt qui contient les données stockées par la collecte fichier.
-Un Makefile.
-Un répertoire src qui contient les sources
-Un répertoire tst qui contient les tests.


Pour compiler et executer il suffit d'executer la commande make.
